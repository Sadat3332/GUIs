# Pokedex
#### This project is a GUI Pokedex application built using Python's Tkinter library and Pandas,It displays detailed information about Pokémon, including their images, descriptions, and base stats.
## Features:
#### Pokemon List: Browse through a list of Pokemon names.

#### Search Functionality: Filter the list of Pokémon by name
<img src="https://github.com/Sadat3332/GUIs/blob/main/Poke/images/Readme/Search.png"  width="40%" height="40%">

#### Pokemon Details: View detailed information including name, type, classification, height, weight, and base stats

## Setup

1. **Clone the repository**:

   ```sh
   git clone https://github.com/Sadat3332/GUIs.git
   cd GUIs
   cd Poke

2. **Install Dependencies**:
- Install Pandas via pip
   ```sh
   pip install pandas
- Tkinter usually comes pre-installed with Python. If you do not have it, you can install it as follows (For Ubuntu):
   ```sh
   sudo apt-get install python3-tk
- Run the app
  ```sh
  python3 pokedex.py
